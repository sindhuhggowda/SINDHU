# Credit Risk Assessment Using Machine Learning

## Project Overview

This project is a machine learning-based Credit Risk Assessment system developed using Python and Scikit-learn. The purpose of this project is to predict whether a loan applicant is classified as a **Low Risk** or **High Risk** customer based on financial and personal information.

Banks and financial institutions use credit risk assessment systems to reduce loan default risks and improve decision-making.

---

## Objective

The main objective of this project is to:

- Analyze customer financial data
- Predict loan repayment risk
- Classify applicants into:
  - Low Risk
  - High Risk
- Improve decision-making using machine learning

---

## Features

- Data preprocessing
- Risk prediction using Machine Learning
- Random Forest Classifier
- Accuracy evaluation
- Classification report generation
- New customer risk prediction

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- NumPy
